// ===== 第1篇 近期 =====
最近掉入了一个名为直子的深井里，持续了几周，被极限的情绪来回支配
两天走了好多路，又两天通宵游戏，让身体和精神双双疲倦，才恢复过来一些。赶上昨天j cole发了准备退休的专辑，我其实被前几首先行拉高了期待，可惜只有一首收录进去了。其实这张专辑不发出来会让人有很多畅想。对他构思了这么多年的退休，我更多的是感慨吧。其实时间过的真的好快呀，两年前我还是个文盲，emmm现在也算吧，但是也已经从只知道名侦探柯南和怪盗基德，从阿加莎等欧美的读到新本格零食杏仁，岛田庄司和社会派了。
川端康成，三岛由纪夫，龙之介等等等等，也算是弥补了从小书读得少的缺点。
这何尝不让人满意呢，可是我越来越孤独了，我知道我读的越多，就越不能聊这些，很少有人愿意听吧。同样读过的人，会有不同的想法，其实更加容易产生冲突。
最后走在路上就只剩下了自己和前辈们，每个人都有自己的道路，那道路上又只有自己。其实我很少发觉我已经这么疲倦了，明明我是最不努力的那个。同学们有的找到了好的工作，有的发了论文，有的做出了喜欢的项目，参加比赛拿了奖。但我确实大学这几年，几乎什么都没做，考试来了准备几天，就这样。对内心世界的探求以牺牲了对现实的探索的话，我觉得被定义成脆弱，敏感，弱者这些都没什么问题。
我被意义困住了太久。我真的很羡慕那些自恋的人，在人这么渺小的物种诞生了这样相悖的情感。
我也羡慕不用思考就可以活得很开心的人，寻找答案的人发现答案出现在不寻找他们的人身上。
我应该会逐渐淡去我的存在，给自己放一个假，在此之前，就再努力的撑一撑好了！

我想了很久的意义，以下也会是很多书里的处理方式：
1.自己赋予意义，感受生活的美好，建造一座空中楼阁，我建了两年，但碎了，无法自欺欺人，就好比真的有人能在熬夜到凌晨五点看到窗外的夜色说，夜色真美，牛马到现在我太开心了！
2.爱，可是在这个时代，爱可能不再持久，如果把意义放在别人身上，终究很难维持，只有在动态的平衡中，爱才能发挥出他的价值，那是多么幸福的体验啊！
3.荒诞主义，燃起生命的焰火，好吧我对这个不太了解，加缪看的不多
4.以某种观念和规则实现超越，到最后会呈现出顽强的生命力，人成为规则的具象化，超人吗，我又不想，我没有在所有人心目中都必须是完美的执念。相反的，我一直在做的都是在诉说我的不完美的地方。会觉得这样更像人一点。
5.虚无主义，躺平吧，摆烂吧，世界上的事情都是没有意义的
所以我的构想变成了不再把现在分割成过去和未来，日本作家很多对于毁灭有着病态的迷恋，很多人在写完了书就自杀了，那他的书和自传+遗书也没太大区别了。我不推崇这种过分的迷恋但对于生于死，毁灭与创造的追求或许是一个解答，到最后生而不有，为而不恃。当然这也只是我的一点点看法。

// ===== 第2篇 挪威的森林读后感 =====
读挪威的森林，我会想有没有可能直子有更好的结局，如果，男主能够放弃掉所有去陪她，如果，她能放弃掉她对完美的执念和残酷的过往。我从一开始就知道了她的结局，所以越往后读越心痛。如果文中没有绿子，没有提到绿子就明显明快的节奏，那不知道还会不会读下去。我觉得作者最温柔的地方是一笔带过了她的离开。没有写她的折磨，她的痛苦，至少之后的描述看她是解脱的。我能理解为什么她的眼神是对世界疏离的，为什么连遗书都没有，能理解她为什么写不了信，其实，她可能后来也看不了书。我看书里评论都在讨论，他们遇到的人像绿子还是直子，红玫瑰和白玫瑰。她掉入了那口深井，而活着的人无法理解，正常人如果无限的接近，就会变成活着的墓碑。有时候追问问题的答案，以为知道了答案就解决了问题。其实，也许是先有了情绪才开始思考这些问题。解决了一个，下一个就出现，当真的没问题想了，悲伤依旧会不讲道理的降临，没有理由，无法反抗。

// ===== 第7篇 11月 =====
11月，转眼又是一年，前年的跨年夜，去年的圣诞恋歌，不知道今年会写什么东西来纪念这一年。去年也是像现在一样，缩在公寓的床上，不知道看了什么，一年就在那之中结束了。节日，生日，这些都离我越来越远。我会因为中秋节想吃月饼，也会因为打折一周后再去买月饼，还会因为卡路里太高放弃月饼。对过去越来越淡漠，也不期待未来的话，我算是活在当下了吗？随着几次考砸，我也知道追求了三年的成绩终究是一场梦。其实我何尝不是期待着有一个理由让自己休息呢。不过是三年而已，也只有处在这个时期才会在乎吧。我也可以开始享受生活，就像比起这学期的成绩，我更在乎今天夹出的发型是不是更满意。当我越急功近利，想要的越得不到。股票账户的数字闪烁着，今天赚了多少，明天赔了多少，又是哪一天一个月的房租就消失了。都是梦幻泡影。投了上百份的简历却没有工作，被不同的人提醒，我也焦虑过，但能让我安心的是寂寥的夜和数不清的星。



// ===== 第8篇 续篇：两个假设 =====
继上次分析生活的意义后，我做了两个假设，一种是戒掉所有的欲望，到一无所有的时候，最先冒出的念头一定是最想得到的东西。但我其实并不能做到苦行僧式的生活。我也不赞成放弃现实所有的压力，责任，排解空虚的日常生活，来面对虚无，没有压力的时候也很难产生欲望。第二个假设是，像金阁寺那样（不能与金阁身心一体的绝望，毁灭才能与他同在），毁掉最喜欢的，产生绝望。毁掉最恨的，得到解脱。但我对毁灭已经，不感兴趣了就像我不会因为我最喜欢的衣服坏了就感到开心。当我对这一切都没有兴趣，我看到身边的人在合理化自己的行为，否定可以是超越的契机，也可以是以否定来逃避借以走向堕落的标志。没有人给我答案，没有书给我答案。罪与罚的结尾把一切归为爱，可是爱是怎样的，他自己都没有写下去。如果跪的是全人类的苦难，那他应该长跪不起。从本格推理读到新本格和社会派，我被华丽的密室征服，被细腻的文字打动，我贪婪的阅读了一本又一本，可我是一个读者，我并不擅长解谜。我也可以接受推理小说来分析人性，但我不接受新本格为了一个诡计建一栋奇怪的房子。日本人总能以他们病态到极致的想象力推陈出新，毁灭，病态，物哀。我为之感慨，我也为之感到恶心。当我学数学，我的这种感觉又被放大了，痴迷于探索他，做出答案的想法。但与之相对的是并不复盘，并不关心方法。只是贪婪的阅读他，享受它。这样的我是在干什么呢。一次又一次进入了看似不同实则相同的谜团。充实只是假象，追逐他也只是离他很远。pyq不该是我发泄的地方，这是我的错误，也有我的私心（就是想说话的时候），但我确实确实会停止发了。



// ===== 第9篇 爱与意义的解构（更早期） =====
爱是很多书中对自我意义的最终解答，但如果一个人得不到爱呢，如果他失去了爱呢，难道为了填满自身就要辗转于爱之间，或者恶俗一点说，见一个爱一个？还是寻找现实中的意义，用纯粹的金钱观代替，但是大多数人好像介于两者之间。那么抹去爱，推及自我呢，自我就是一切的最中心，但是这样经不起解构，解构到最后，逻辑，思考，自我，一切都会被推到无意义和不存在。在没有任何意义的世界上活着，怎样活才有意义呢，这是曾困住我很久的问题。自己赋予意义这种东西和欺骗自己没有区别。思考的越多并不代表越透彻，他只是代表了更多的可能性以及在这种可能中钻研的更深，但这种可能本身可能也只是一个自欺欺人的循环。所以我不再追求意义，不再构建意义，我不去寻找任何方向，我想要解构我自己，哪怕到最后发现只是一个空壳。

看到了你的尽头，你是偏执的改变世界，随波逐流，还是抹掉你的爱恨？
都走的那么快，眼神僵直，我仿佛看到了我未来的样子，我为什么，还是慢不下来？我还有多久的自由？
我的病快好了但我没承认过我有病。束缚一个个都解开了。
我的一切都在回来，那我就又要选了，两个都是错的，你们相信"存在"吗？这一回，我就再选个错的吧。。。

// ===== 第49篇 命运齿轮（古风短篇） =====
命运齿轮悄息转动，霓虹辉映，残虹角落，弹指芳华，已逝万年。明月皎皎，瘦月寥寥，琴音萧萧，余音渺渺。暖色帷幔，烈日炎炎，光映眼内，目之所及，心之所念，青丝红颜，如梦似幻，醉梦仙霖。心软的没了力气跳，暖阳下，你舞姿翩翩在笑，寒夜里，我醉梦酣欢心在烧，漂泊有了羁绊，羁旅不再孤单，爱，是氤氲香气的一缕，是回眸泪湿瞳孔的感动，是千年流转，载君一意。

// ===== 第50篇 醒着像睡着（歌词/散文诗，最早期） =====
醒着像睡着，躺着想思考，晨昏颠倒。
生活太单调，学习多纷扰，跟谁比较？
如梦似醒，欲眠似醉，多纷扰？
我也累了，她也笑了，是刚好(✪▽✪)
在风中清醒，于雨中睡去，多半是梦境，关心却犹豫，浪漫的气息，在围绕。
也许人走茶凉散落一地忧伤，也许干戈铁马落下一阵荒凉。
但你还是你，风飘忽千里，不明归期，雨打在夜里，无人在意，可我，如落叶飘忽数息，芳华美丽，如那年樱花，载君一意。
对的时间点，多看你一眼，对的时间线，幸会能遇见。
你们悔恨当初，你们怀恋初恋，你们一曲断肠，你们心碎魂殇，你们放下妆容，你们裹上新装，你们苍老不已，听见婴儿哭啼，你们相视一笑，然后满眼泪滴。
是谁的爱呀，欲言又止的悲伤，是谁的爱呢，青春嘻戏的荒唐，是谁的爱呵，来的那样突然又慌张，又是谁的爱啊，敏感的夸张，脆弱的疯狂。
我愿安守一隅岁月的静谧，感叹时间静好。我愿行遍天下风景，夸耀鬼斧神工。
我愿看尽世间繁华落尽，赞美本色的荒凉。
我愿看清，一条条红线的交织，一对对爱情的烂漫，一束束玫瑰，一个个画圈，起于动，归于静，最后魂归冥冥。
